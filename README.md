# Firebase Web Hosting
## Global Cli
Open CMD and type 
```bash
npm install -g firebase-tools
```

## Login
```bash
firebase login
```

## Go to project location
Builds > FirebaseBuild 

## Link Firebase project to local directory
```bash
firebase projects:list
firebase use --add
```

## Local Hosting
```bash
firebase emulators:start
```

## Live Deploy
```bash
firebase deploy
```
